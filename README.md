# Velocity-CPP

An industry-grade game engine written for C++. The base engine files will be hosted here, with extension modules that can optionally be linked in.

## Quickstart (Linux)
```
$ mkdir build
$ cd build
$ cmake ..
$ make
```

## Quickstart (Windows + MSYS2)
```
PS> mkdir build
PS> cd build
PS> cmake .. -G "MSYS Makefiles"
PS> make
```
